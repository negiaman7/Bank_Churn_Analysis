This project focuses on analyzing customer churn behavior in the banking sector using Power BI. The dashboard provides a complete view of customer activity, churn patterns, financial impact, and retention opportunities from a dataset of 10,000 customers. Customer churn is one of the biggest challenges in the banking sector. This project analyzes a dataset of 10,000 customers to identify why customers leave, which segments are most at risk, and how churn impacts business revenue.
The dashboard supports decision-makers with insights into customer behavior, financial loss, and targeted retention strategies.
